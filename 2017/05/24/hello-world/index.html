XMLHttpRequest详解

对象创建XMLHttpRequest

兼容判断
var xhr;
if (window.XMLHttpRequest)
{
// IE7+, Firefox, Chrome, Opera, Safari 浏览器执行代码
xhr=new XMLHttpRequest();
}
else
{
// IE6, IE5 浏览器执行代码
xhr=new ActiveXObject(“Microsoft.xhr”);
}

get请求步骤

  • var xhr = createXHR();//创建对象
  • xhr.open(‘get’, ‘03form.php’, true);//初始化请求
  • xhr.send(null);;//发送请求
  • xhr.onreadystatechange = function() {}//指定一些回调函数
  • 状态值xhr.readyState

    post请求步骤

  • xhr.open(“post”,”地址”,”标志位”);//初始化请求
  • xhr.setRequestHeader(“”,””);//设置http头信息
  • xhr.send(参数);//发送请求
  • xhr.onreadystatechange = function() {
    //请求成功
    if(xhr.readyState == 4){
    if(xhr.status == 200){
    //取回返回数据
}else{
    //请求失败执行代码
}
    }
    }//指定回调函数

    xhr.readyState详解

  • 0:XMLHttpRequest 对象创建完成
  • 1：表示发送请求的动作准备好了，但是还没有开始发送
  • 2：表示已经发送完成
  • 3：服务器已经返回了数据
  • 4：服务器返回的数据已经可以使用

    ajax跨域问题

    出于安全性考虑，浏览器不允许ajax跨域获得数据
    跨域通过script的src加载js的方式传递数据以函数参数的形式传统数据
    解决方法：就是在请求的API中加入要请求的跨域的网址，当请求成功后，将返回的网址动态的加载到页面上，就可以解决跨域问题
